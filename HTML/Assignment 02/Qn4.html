1. <html> – The root element of an HTML document. It wraps all other content on the page.
  2. <head> – Contains meta-information about the document, such as title, character encoding, linked stylesheets, and scripts.
  3. <title> – Sets the title of the web page, which appears in the browser tab.
  4. <meta> – Provides metadata such as description, keywords, or viewport settings. It is a self-closing tag.
  5. <link> – Used to link external resources like CSS files. It's also self-closing.
  6. <body> – Contains the visible content of the webpage, such as text, images, and elements.
  7. <h1> to <h6> – Header tags used to define headings, where <h1> is the most important and <h6> the least.
  8. <p> – Represents a paragraph of text.
  9. <a> – Defines a hyperlink, allowing users to click and navigate to another page or resource.
  10. <img> – Embeds an image in the page. It is a self-closing tag and requires a src and alt attribute.
  11. <div> – A generic container used for grouping and styling content using CSS or JavaScript.
  12. <span> – An inline container used to style a part of text or other inline elements.
  13. <ul> – Creates an unordered (bulleted) list.
  14. <ol> – Creates an ordered (numbered) list.
  15. <li> – Represents a list item in <ul> or <ol>.
  16. <table> – Defines a table.
  17. <tr> – Defines a table row.
  18. <td> – Defines a table cell (data cell).
  19. <th> – Defines a table header cell.
  20. <form> – Used to collect user input with fields like textboxes, checkboxes, etc.
