1. HTML

    Tech Stack Primary Use Cases Key Features / Benefits
    HTML
    • Structuring web pages
    • Defining headings, paragraphs, links, etc.
    • Embedding media (images, videos)
    • Easy to learn and write
    • Foundation of all web pages
    • Works with all browsers

  2. CSS

    Tech Stack Primary Use Cases Key Features / Benefits
    CSS
    • Styling HTML elements
    • Creating layouts and responsive designs
    • Animations and visual effects
    • Separation of content and style
    • Responsive design with media queries
    • Supports animations and transitions

  3. JavaScript

    Tech Stack Primary Use Cases Key Features / Benefits
    JavaScript
    • Making web pages interactive
    • Form validation
    • Fetching and displaying data dynamically
    • Runs in all modern browsers
    • Supports event handling and dynamic content
    • Used on both front-end and back-end (Node.js)
