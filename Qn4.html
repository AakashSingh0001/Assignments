1. Use Semantic HTML Elements

3. Ensure Keyboard Navigability

Make sure users can navigate the site using only the keyboard (e.g., Tab, Enter, Arrow keys). Use tabindex, and make interactive elements (like buttons and links) focusable and usable without a mouse.
